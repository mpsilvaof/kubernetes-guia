# Segurança

Boas práticas de segurança em ambientes Kubernetes:

- Usar **RBAC (Role-Based Access Control)**.
- Evitar rodar contêineres como root.
- Aplicar **Network Policies**.
- Escanear imagens antes do deploy.
- Manter o cluster e componentes atualizados.

Referência: [Kubernetes Security](https://kubernetes.io/docs/concepts/security/)
