# Networking no Kubernetes

O modelo de rede do Kubernetes garante comunicação entre Pods e Services.

## Principais Conceitos
- **ClusterIP:** comunicação interna.
- **NodePort:** acesso externo básico.
- **LoadBalancer:** integração com balanceadores de nuvem.
- **Ingress Controller:** controle de entrada HTTP/HTTPS.

Referência: [Kubernetes Networking](https://kubernetes.io/docs/concepts/services-networking/)
