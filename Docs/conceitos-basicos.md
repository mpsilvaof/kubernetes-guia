# Conceitos Básicos

Kubernetes é uma plataforma open-source para orquestração de contêineres, originalmente desenvolvida pelo Google.

## Principais Componentes
- **Pod:** menor unidade de execução no Kubernetes.
- **Node:** máquina física ou virtual que executa os pods.
- **Cluster:** conjunto de nodes gerenciados por um plano de controle.
- **Deployment:** gerencia réplicas e atualizações de pods.
- **Service:** expõe aplicações dentro ou fora do cluster.
- **Ingress:** gerencia o acesso HTTP/HTTPS.

Referência: [Kubernetes Concepts](https://kubernetes.io/docs/concepts/)
